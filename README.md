IMAP Server Tester
==================

ImapTest is a generic IMAP server compliancy tester that works with all IMAP servers. It supports:

 - Stress testing with state tracking
 - Scripted testing
 - Benchmarking

Please see https://dovecot.github.io/imaptest/ for more information.

Running with Docker
-------------------

This tool is also available on Docker, please see https://hub.docker.com/r/dovecot/imaptest for more information.

To run, use `docker run --rm -it dovecot/imaptest`.
